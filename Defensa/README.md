## Descripción
Este repositorio contiene la resolución de los tres planteamientos de la prueba diagnóstica:
1. **Analizador Léxico:** Clasificación de tokens (NUMERO, OPERADOR, etc.) y balance de paréntesis.
2. **Validación FEN:** Verificación de cadenas en notación Forsyth-Edwards para ajedrez.
3. **Conjetura de Collatz:** Algoritmo aplicado a un intervalo $[p, q]$ bajo la regla $q \ge 100p$.

## Estructura
El proyecto sigue la estructura de carpetas exigida en la ponderación:
- `/Defensa`: Documento con el enlace a la explicación audiovisual.
- `/Problema1`: Código fuente del lexer.
- `/Problema2`: Lógica de validación FEN.
- `/Problema3`: Implementación de Collatz.


## Ejecución
Para ejecutar los scripts, asegúrese de tener instalado **Node.js**.

# Para el Problema 1
node problema1/analisisLexico.js

# Para el Problema 2
node problema2/notacionFEN.js

# Para el Problema 3
node problema3/conjeturaDeCollatz.js
